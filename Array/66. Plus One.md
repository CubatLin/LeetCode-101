### 原始思路
```
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        # 直覺的思路是9就要進位/其他不用, 然後從尾端開始加
        tmp = 0; start = True
        for i in range(len(digits)-1, 0, -1):
            if digits[i] == 9:
                # 進位, 把digits[i]改成0, 然後讓tmp +=1
                tmp = 1; digits[i]=0
            else: 
                # 不進位, 把tmp加到這個位數, tmp 重新歸0
                if start:
                    digits[i] = digits[i] + tmp + 1
                    tmp = 0
                    start=False
        # 最後回傳digits
        return digits
```
- 錯誤點:
    - 多用tmp跟start
    - 沒有處理[9]+1 > [1,0]
    - range錯誤: range(len(digits)-1, -1, -1)

### GPT修正:
- 其實不用想tmp, 這個問題本質是這個位數有沒有進位(if-else), 如果加到不是9的位數+後可以直接return
- 重點是就算遇到9, 也直接改0, 進位的部分其實跟原始條件要+1是一樣邏輯 
```
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        # 直覺的思路是9就要進位/其他不用, 然後從尾端開始加
        for i in range(len(digits)-1, -1, -1):
            if digits[i] == 9:
                # 進位, 把digits[i]改成0
                digits[i] = 0
            else: 
                # 不進位, 把tmp加到這個位數, tmp 重新歸0
                digits[i] += 1
                return digits
        # edge case: 如果是[9] > [1, 0]
        return [1] + digits
```
