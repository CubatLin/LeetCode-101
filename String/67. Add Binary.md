
## 思路
- 要把一個數字拆解成十進位，每次就是「除以 10 看餘數」：
    - 123 ÷ 10 = 12 ... 3 → 個位數是 3
    - 12 ÷ 10 = 1 ... 2 → 十位數是 2
    - 1 ÷ 10 = 0 ... 1 → 百位數是 1
- 二進制就是/2看餘數
    - 7/2 = 3 餘 1
    - 3/2 = 1 餘 1
    - 1/1 = 0 餘 1
    - 7 = '111s'
- 實現:
```
def toBinary(num: int) -> str:
    res = ''
    # 特殊情況：0
    if num == 0:
        return "0"

    # 當商數不是 0，就一直除以 2
    while num != 0:
        remainder = num % 2             # 取餘數
        res = str(remainder) + res      # 餘數加在 res 前面（因為低位在右邊）
        num //= 2                       # 更新 num = 商
    return res
```
